# Ray-Shade

## Features :

- User can create, update, delete and sell product
- Product can e filtered using various properties
- Bulk delete supported
- User can not access anything without login
- To ensure security jwt token has been used

## Technology Used :

- ReactJs
- Redux
- ShadCn
- React Router Dom
- zod
- React Hook Form
